---------------------------------------------------------------------------------------

Display [Taskwarrior](https://taskwarrior.org/)
 task data for configurable commands. Each named command runs `task` with the specified arguments and exposes the raw output in the template.

Sample Configuration[​](https://ohmyposh.dev/docs/segments/cli/taskwarrior#sample-configuration "Direct link to Sample Configuration")

Options[​](https://ohmyposh.dev/docs/segments/cli/taskwarrior#options "Direct link to Options")

------------------------------------------------------------------------------------------------

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `command` | `string` | `task` | the Taskwarrior executable to use |
| `commands` | `map[string]string` | see below | map of name to Taskwarrior arguments; the raw output is exposed in `.Commands` |

### Default `commands` value[​](https://ohmyposh.dev/docs/segments/cli/taskwarrior#default-commands-value "Direct link to default-commands-value")

    {  "due": "+PENDING due.before:tomorrow count",  "scheduled": "+PENDING scheduled.before:tomorrow count",  "waiting": "+WAITING count",  "context": "_get rc.context"}

Each entry runs `task <arguments>` and stores the trimmed stdout as a string. Remove entries you do not need to keep prompt rendering fast.

Template ([info](https://ohmyposh.dev/docs/configuration/templates)

default template

     {{ "\uf4a0" }} {{ range $k, $v := .Commands }}{{ $k }}:{{ $v }} {{ end }}

### Properties[​](https://ohmyposh.dev/docs/segments/cli/taskwarrior#properties "Direct link to Properties")

| Name | Type | Description |
| --- | --- | --- |
| `.Commands` | `map[string]string` | raw command output keyed by name with the first letter uppercased (e.g. `"Due"`) |

### Examples[​](https://ohmyposh.dev/docs/segments/cli/taskwarrior#examples "Direct link to Examples")

Access a specific command result directly:

     {{ "\uf4a0" }} Due: {{ .Commands.Due }}

Display multiple results:

     {{ "\uf4a0" }} Due: {{ .Commands.Due }} | Waiting: {{ .Commands.Waiting }}

Include the active context alongside task counts:

     {{ "\uf4a0" }} {{ .Commands.Context }} - Due: {{ .Commands.Due }} Scheduled: {{ .Commands.Scheduled }}
