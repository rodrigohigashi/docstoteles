What[​](https://ohmyposh.dev/docs/segments/music/lastfm#what "Direct link to What")

------------------------------------------------------------------------------------

Show the currently playing song from a [LastFM](https://www.last.fm/)
 user.

caution

Be aware that LastFM updates may be severely delayed when paused and songs may linger in the "now playing" state for a prolonged time.

Additionally, we are using HTTP requests to get the data, so you may need to adjust the `http_timeout` to your liking to get better results.

You **must** request an [API key](https://www.last.fm/api/account/create)
 at the LastFM website.

Options[​](https://ohmyposh.dev/docs/segments/music/lastfm#options "Direct link to Options")

---------------------------------------------------------------------------------------------

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `playing_icon` | `string` | `\uE602` | text/icon to show when playing |
| `stopped_icon` | `string` | `\uF04D` | text/icon to show when stopped |
| `api_key` | [`template`](https://ohmyposh.dev/docs/configuration/templates) | `.` | your LastFM [API key](https://www.last.fm/api/account/create) |
| `username` | `string` | `.` | your LastFM username |
| `http_timeout` | `int` | `20` | in milliseconds - the timeout for http request |

### Properties[​](https://ohmyposh.dev/docs/segments/music/lastfm#properties "Direct link to Properties")

| Name | Type | Description |
| --- | --- | --- |
| `.Status` | `string` | player status (`playing`, `paused`, `stopped`) |
| `.Artist` | `string` | current artist |
| `.Track` | `string` | current track |
| `.Full` | `string` | will output `Artist - Track` |
| `.Icon` | `string` | icon (based on `.Status`) |
