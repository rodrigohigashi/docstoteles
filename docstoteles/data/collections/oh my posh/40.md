What[​](https://ohmyposh.dev/docs/segments/cli/gitversion#what "Direct link to What")

--------------------------------------------------------------------------------------

Display the [GitVersion](https://github.com/GitTools/GitVersion)
 version. We _strongly_ recommend using [GitVersion Portable](http://chocolatey.org/packages/GitVersion.Portable)
 for this.

caution

The GitVersion CLI can be a bit slow, causing the prompt to feel slow. This is why we cache the value for 30 minutes by default.
